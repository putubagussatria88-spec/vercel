Server-Side Fixes & Mitigations SELECT YOUR STACK
Critical
1 · Upgrade to patched build
Nginx mainline ≥ 1.27.3 or stable ≥ 1.26.3. Verify with nginx -v.
# Ubuntu/Debian
sudo apt update && sudo apt install --only-upgrade nginx

# Verify HTTP/2 module present
nginx -V 2>&1 | grep http_v2
2 · Harden nginx.conf
http {
  # Reduce concurrent streams (default 128 is exploitable)
  http2_max_concurrent_streams 32;
  http2_idle_timeout           30s;
  http2_body_preread_size      16k;
  http2_max_requests           1000;
  http2_recv_timeout           15s;
  http2_max_field_size         4k;
  http2_max_header_size        16k;

  # Per-IP connection limit
  limit_conn_zone $binary_remote_addr zone=addr:10m;
  server {
    limit_conn        addr 10;
    limit_conn_status 429;
  }
}

2 · RST_STREAM tracking in plain Node http2
const http2 = require('http2');
const server = http2.createSecureServer({ settings: {
  maxConcurrentStreams: 32,
  initialWindowSize: 65535,
  maxHeaderListSize: 8192
}});

server.on('session', session => {
  let rst = 0, win = setTimeout(() => rst = 0, 10000);
  session.on('stream', stream => {
    stream.on('close', () => {
      if (stream.rstCode !== 0 && ++rst > 100)
        session.destroy(http2.constants.NGHTTP2_ENHANCE_YOUR_CALM);
    });
  });
  session.on('close', () => clearTimeout(win));
});

Vercel Deploy
Prerequisites
Node.js ≥ 20, npm ≥ 10, a free Vercel account at vercel.com
1 · Install Vercel CLI
npm install -g vercel
2 · Clone & install dependencies
# Download or unzip the project folder, then:
cd madeyoureset-detector
npm install
3 · Test locally
vercel dev
# Opens http://localhost:3000
# API available at http://localhost:3000/api/probe
4 · Deploy to production
vercel --prod
# First run: creates project, links to your Vercel account
# Subsequent runs: instant redeploy
# Output: https://your-project.vercel.app
5 · Optional — set environment variables
# In Vercel dashboard → Settings → Environment Variables
ALLOWED_ORIGIN=https://yourdomain.com

# Or via CLI:
vercel env add ALLOWED_ORIGIN
6 · Add custom domain
vercel domains add yourdomain.com
# Then point your DNS CNAME to cname.vercel-dns.com

Anti-DDoS layers active on /api/probe
All requests to /api/probe pass through: Global burst limiter (1000 req/s) → IP block check → Per-IP rate limit (30 req/min) → Per-IP token bucket (burst 10) → Anomaly scoring (UA, headers, path) → RST_STREAM probe detector → Body size limit (4KB) → SSRF prevention (no private IPs).
